# ✅ БОТ ПРАЦЮЄ! Quick Start Guide

## 🎉 Ваш Telegram бот з inline кнопками готовий!

---

## 🚀 ЯК ЗАПУСТИТИ (2 кроки):

### 1️⃣ Запустіть dev server:

```bash
npm run dev
```

### 2️⃣ Запустіть бот (в новому терміналі):

```bash
npm run bot
```

### 3️⃣ Відкрийте Telegram:

Знайдіть бота: **@OneCompanyAutoBot**

Надішліть: **/start**

## ✨ ФУНКЦІЇ БОТА:

### 🏆 Наші бренди
- Вибір категорії (Авто/Мото)
- Список преміум брендів
- Кнопка "Залишити заявку"
- Посилання на повний каталог

### 📱 Контакти
- Email, телефон, адреса
- Графік роботи
- Кнопка назад

### 🌐 Web3 Dashboard
- Посилання на http://localhost:3001/messages
- Інструкції по авторизації
- Список можливостей

### ❓ Допомога
- Інструкції як замовити
- Кнопки швидкої заявки (Авто/Мото)

### 🚗/🏍️ Заявки
- Форма для замовлення
- Автоматична пересилка менеджеру
- Збереження в Dashboard
