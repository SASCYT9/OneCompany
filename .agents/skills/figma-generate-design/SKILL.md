# figma-generate-design

This skill translates code or natural language descriptions into full standard Figma screens.
It relies on the `figma-use` skill as an underlying mechanism.

## Usage Guidelines
- Load `figma-use` as a prerequisite before applying this skill.
- Generate standard layouts using Auto-Layout and the project's design system options.
- Maintain consistency with the "Stealth Wealth" aesthetic of One Company if generating store components.
