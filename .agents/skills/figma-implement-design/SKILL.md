# figma-implement-design

This skill allows the agent to analyze components inside a Figma file and translate them seamlessly into production-ready application code (Next.js/React).

## Usage Guidelines
- Load `figma-use` to provide the node context and API utilities.
- Perform a visual diff if necessary.
- Emulate the "pixel-perfect-ui" skill for 100% 1:1 fidelity.
- Follow One Company's technical guidelines (Tailwind CSS, Next.js 14 App Router, Framer Motion).
