# Premium Web Design Skill

Expert guidance for creating "Apple-quality" premium digital experiences. Focuses on restraint, high-end aesthetics, and psychological comfort.

## Core Directives

### 1. Typography & Hierarchy
- **Restraint:** Never use more than two font families. Prefer variable fonts with wide weight ranges.
- **Micro-typography:** Pay obsessive attention to kerning, tracking (-0.02em for headers), and line-height (1.5-1.6 for body, 1.1-1.2 for headers).
- **Scale:** Use a modular scale (e.g., 1.25x) to ensure mathematical harmony between text sizes.

### 2. Spacing & Rhythm (The "Breathability" Principle)
- **Generous Whitespace:** Use 2x more whitespace than you think is necessary. Space is a luxury indicator.
- **Soft Grid:** Align everything to an 8px grid, but allow for "optical center" adjustments.
- **Sectioning:** Use large vertical padding (120px-200px) between major sections to allow each idea to breathe.

### 3. Color & Texture
- **Subtle Palettes:** Prefer high-contrast backgrounds (Deep Black #000 or Pure White #FFF) with subtle semantic accents.
- **Layering:** Use semi-transparent surfaces (glassmorphism) with background blurs (20px-40px) to create depth.
- **Gradients:** Only use "mesh" or extremely subtle linear gradients (2-3% difference in color).

### 4. Motion & Interaction
- **Ease-in-Out:** Use custom cubic-bezier curves (e.g., `cubic-bezier(0.4, 0, 0.2, 1)`) for all transitions. No "linear" motion.
- **Staggered Reveals:** Elements should appear in a sequence, not all at once.
- **Micro-interactions:** Add haptic-like visual feedback on hover/click (subtle scale down to 0.98, subtle glow).

### 5. Premium Content
- **Imagery:** Use high-resolution, professionally shot photography. Prefer studio lighting or high-contrast outdoor shots.
- **Copy:** Use "Less, but better" approach. Every word must justify its existence. Avoid marketing fluff.

## Design Evaluation Checklist
- [ ] Is the focal point clear within 1 second?
- [ ] Does the layout feel balanced but not necessarily symmetrical?
- [ ] Are the interactive elements clearly distinguishable without being "loud"?
- [ ] Does the motion feel "heavy" (deliberate) rather than "fast" (cheap)?
- [ ] Is there enough "negative space" to evoke a sense of premium quality?
