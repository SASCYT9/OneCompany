# figma-create-design-system-rules

This skill establishes and maintains the design system rules mapped between Figma and the codebase.

## Usage Guidelines
- Document standard Tailwind classes, color palettes ("Stealth Wealth"), typography, and spacing used by One Company.
- Ensure any generated Figma designs adhere to these rules via the `figma-use` linter options.
