# figma-create-new-file

This skill provides instructions on creating a new Figma file and initializing its basic parameters before design components are generated.

## Usage Guidelines
- Utilize Figma's REST API or `figma-use` to instantiate a new draft or project file.
- Add foundational tokens (via `figma-use`'s `defineComponent` or variables) mimicking the project's Tailwind config upon creation.
