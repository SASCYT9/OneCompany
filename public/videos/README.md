# Відео для OneCompany 3D

Для повноцінної роботи сайту потрібно додати відео файли у цю папку.

## Необхідні відео:

### 1. KW Suspension (`kw-suspension.mp4`)
- **Контент**: Відео з реальними продуктами KW Suspension
- **Рекомендації**: 
  - Показати койловери KW V3 у дії
  - Демонстрація регулювань підвіски
  - Кадри авто на треку з КW підвіскою
  - Крупні плани німецького інжинірингу
- **Формат**: MP4, H.264
- **Розмір**: 1920x1080 (Full HD) мінімум
- **Тривалість**: 10-30 секунд (зациклене)
- **Розмір файлу**: оптимально до 5-10MB для швидкого завантаження

### 2. Fi Exhaust (`fi-exhaust.mp4`)
- **Контент**: Відео з реальними титановими вихлопними системами Fi
- **Рекомендації**:
  - Титанові труби великим планом
  - Демонстрація Valvetronic системи
  - Авто з Fi Exhaust в русі
  - Звук вихлопу (якщо можливо)
  - Крупні плани якості виробництва
- **Формат**: MP4, H.264
- **Розмір**: 1920x1080 (Full HD) мінімум
- **Тривалість**: 10-30 секунд (зациклене)
- **Розмір файлу**: оптимально до 5-10MB

### 3. Eventuri Intake (`eventuri-intake.mp4`)
- **Контент**: Відео з карбоновими впускними системами Eventuri
- **Рекомендації**:
  - Карбонові деталі великим планом
  - Демонстрація патентованої технології Venturi
  - Процес встановлення
  - Авто з Eventuri в дії
  - Акцент на преміальності та аеродинаміці
- **Формат**: MP4, H.264
- **Розмір**: 1920x1080 (Full HD) мінімум
- **Тривалість**: 10-30 секунд (зациклене)
- **Розмір файлу**: оптимально до 5-10MB

## Загальні вимоги:

- **Якість**: Професійне відео з хорошим освітленням
- **Колірна гама**: Темні тони, преміальна атмосфера
- **Стабілізація**: Плавні, стабілізовані кадри
- **Зацикленість**: Відео має бути зациклене без помітних переходів
- **Оптимізація**: Стиснення з балансом якості/розміру
- **Автоплей**: Відео буде автоматично відтворюватись без звуку

## Де взяти відео:

1. **Офіційні канали брендів**: YouTube, Instagram, офіційні сайти
2. **Stock відео**: Artgrid, Storyblocks, Envato Elements
3. **Кастомне відео**: Замовити професійну зйомку продуктів
4. **UGC контент**: З дозволом власників авто з цими компонентами

## Альтернатива (тимчасова):

Якщо поки немає оригінальних відео, можна використати:
- Високоякісні placeholder відео преміум автомобілів
- Абстрактні відео з автомобільною тематикою
- Градієнтні анімації у відповідних кольорах брендів
