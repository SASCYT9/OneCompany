# Vehicle make logos

The PNG make marks are local optimized copies from the MIT-licensed
`filippofilip95/car-logos-dataset` repository. The original logos remain trademarks of their
respective owners.

Exceptions:

- `ducati.svg`: Wikimedia Commons `Logo Ducati.svg` (public-domain text logo).
- `ineos.png`: Wikimedia Commons `Ineos Grenadiers Logo.png` (CC0).
- `ldv.png`: the related SAIC Maxus mark supplied by the dataset because the fitment source uses
  the legacy `Ldv` manufacturer name.
- `maruti.png`: the Suzuki mark supplied by the dataset because the fitment source uses `Maruti`.
- `Range Rover`: currently uses the Land Rover make mark from the same dataset.
