🔴 Можна Робити Прямо Зараз
1. Burger — Фіксовані Розміри Коробки
~15 хв
Всі RaceChip чіпи їдуть в однаковій коробці. Захардкодити weight/dimensions для бренду Burger в ShopBrandLogistics.
→ Точний розрахунок доставки для всіх Burger товарів
2. Turn14 Shipping Quote API
2-3 години
Endpoint POST /v1/quote дає реальну вартість доставки від Turn14/виробника до складу. Функція fetchTurn14ShippingQuote() в turn14.ts. Часткова залежність: потрібна адреса складу (вводиться в адмінці).
→ Реальна вартість inbound shipping замість фіксованих ставок
3. CSV Import Wizard (Phase E)
~1 день
Upload → Column mapping → Dry-run validation → Commit → Error report. Підтримка шаблонів по постачальнику. On conflict: skip/update/create по SKU.
→ Готовність для Atomic CSV, Eventuri та будь-яких інших постачальників
