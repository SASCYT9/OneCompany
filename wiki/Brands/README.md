markup-формули, переклади.

## Шаблон нотатки по бренду
Створюй файл `wiki/Brands/<BrandName>.md` з секціями:
- **Джерело даних** — звідки беремо ціни/описи/фото.
- **Особливості скрейпінгу** — підводні камені (CAPTCHA, lazy-load, мови).
- **Markup-формула** — як рахується ціна (якщо є).
- **Знані баги** — що не так працює і чому.
