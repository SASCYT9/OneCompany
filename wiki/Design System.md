# 🎨 Дизайн Система (Design System)

Проект One Company використовує концепцію преміального дизайну, яка називається **"Stealth Wealth"**. Основний акцент робиться на стриманій розкоші, глибоких темних кольорах, та вишуканих анімаціях. 

Всі стилі та графічні конвенції зафіксовані в `tailwind.config.ts` та глобальних стилях. Не рекомендується використовувати стандартні, "дешеві" або яскраві кольори (наприклад, чистий червоний чи синій) без нагальної брендової потреби.

## 1. Кольорова Палітра (Color Palette)

Конфігурація `tailwind.config.ts` впроваджує спеціальні токени для UI:

### ⬛ Основні темні тони (Obsidian)
- **Obsidian DEFAULT**: `#030303` — супер-чорний, використовується як основний фон (Background) всього додатку. Найтемніший з усіх.
- **Obsidian Light**: `#0a0a0c` — використовується для легкого виділення контейнерів або карток на чорному фоні.
- **Obsidian Panel**: `#0d0d12` — використовується для панелей управління, хедера та футера. Дає глибину.
- **Obsidian Border**: `rgba(255, 255, 255, 0.06)` — ледь помітний, дуже тонкий білий контур для карток та розмежувань. Покращує читабельність, але зберігає преміальність.

### 🟡 Акцентні тони (Bronze)
- **Bronze DEFAULT**: `#c29d59` — основний акцентний колір (кнопки "Buy Now", іконки, статуси).
- **Bronze Light**: `#d4b77e` — варіант для ховер-ефектів (`hover:text-bronze-light`).
- **Bronze Glow**: `rgba(194, 157, 89, 0.15)` — використовується для м'якої тіні (`box-shadow`) або підсвічування довкола кнопок чи карток.

## 2. Анімації та Взаємодія (Animations)

One Company масово використовує бібліотеки для плавних взаємодій:
- **Framer Motion**: Застосовується для мікро-анімацій, плавних переходів сторінок, появи списків (stagger effects). Ніщо в інтерфейсі не повинно з'являтися миттєво.
- **GSAP (GreenSock)**: Використовується для складних скрол-анімацій (scroll-triggered animations), паралаксів та послідовних розгортань контенту.
- **React Three Fiber (@react-three)**: Інтеграція 3D елементів, моделей автозапчастин чи автомобілів безпосередньо у браузер з використанням WebGL. Доповнюється бібліотеками Leva та Valtio для керування стейтом.
- **Lenis (@studio-freight/lenis)**: Відповідає за `smooth scroll` по всьому сайту. Скролінг відчувається як "масло".

## 3. Типографіка (Typography)

У дизайні використовуються преміальні шрифти (Google Fonts/Custom), які підключаються через CSS-змінні:
- **Sans (`var(--font-stack-sans)`)**: За замовчуванням використовується для текстів описів, таблиць та параграфів. Повинен мати гарний кернінг та читабельність.
- **Display (`var(--font-stack-display)`)**: Використовується для великих заголовків (H1, H2) таHero-секцій. Геометричний гротеск або елегантний шрифт з засічками (залежно від філіалу, наприклад Eventuri чи Urban).
- **Mono**: Використовується для SKU моделей автомобілів або таблиць технічних характеристик.

## 4. UI Патерни та Glassmorphism

- **Glassmorphism**: На сайті обережно застосовуються ефекти розмиття заднього фону. Замість непрозорих блоків часто використовуються блоки типу `bg-white/5 backdrop-blur-xl border border-obsidian-border`. Це створює ефект темного скла над контентом.
- **Radix UI / Shadcn**: Запозичено структуру для створення Accordion-меню, Select-ів та інших інтерактивних компонентів (наочний приклад: анімації `accordion-down` та `accordion-up` в Tailwind конфігу).
- **Radical Responsiveness**: Система побудована з брейкпоінтами `xs (475px)`, `sm`, `md`, `lg`, `xl`, `2xl`. UI повинен ідеально лягати на iPhone та великі Ultrawide монітори.

## 5. Графічні Матеріали (Imagery)
- Згідно з протоколом проекту, **всі зображення повинні бути реальними**. 
- Заборонено використовувати стокові картинки-заглушки. 
- AI використовується тільки для генерації фонів/текстур (напр. карбон, метал, асфальт).
- Всі ілюстрації мають оброблятись через CDN (наприклад, Cloudinary) для webp/avif компресії.
