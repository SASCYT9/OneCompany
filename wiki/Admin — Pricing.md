## Функціонал

- **Мультивалютні ціни** (UAH/EUR/USD) для кожного варіанту
- **B2B/B2C price rules** — окремі ціни для різних груп
- **Масове оновлення** — percent markup або fixed amount
- **Курси валют** — синхронізація з НБУ API
- **Regional pricing** — % або fixed корекція по країнах
