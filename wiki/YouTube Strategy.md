# 🎬 YouTube Strategy

> [!info] One Company YouTube — технічний авто-тюнінг контент з editorial якістю
> Частота: **2-4 відео/місяць** (long-form) + **4-8 Shorts/місяць**

---

## 🎯 Позиціонування Каналу

**"Тюнінг, яким він має бути"** — не гаражний блог, а продакшн-якість контент про преміум авто-тюнінг.

### Чим відрізняємось
- 🎥 **Cinematic** — кожне відео = міні-фільм
- 📊 **Data-driven** — dyno numbers, thermal tests, real results
- 🌍 **Multilingual** — UA audio + EN subtitles (або навпаки)
- 🛒 **Shoppable** — links на наші товари в description

---

## 📋 Формати Відео

### 1. 🎬 Build Reveal / Full Review (1-2x/місяць)
**Тривалість:** 8-15 хвилин
**Що:** Повний огляд тюнінг-проекту або продукту
**Структура:**
```
0:00 — Hook (результат спочатку)
0:30 — Intro + контекст
2:00 — Процес / установка
5:00 — Результати (dyno, sound, temps)
8:00 — Висновки + ціна
9:00 — CTA (link in description)
```

**Приклади:**
- "BMW M3 G80 — DO88 Intercooler + RaceChip GTS Black | Повний тюнінг"
- "Brabus G900 Rocket Edition | Urban Automotive Build"
- "Akrapovic Evolution vs OEM Exhaust | Mercedes C63 AMG Sound Test"

### 2. 📊 Comparison / Test (1x/місяць)
**Тривалість:** 6-10 хвилин
**Що:** Об'єктивне порівняння продуктів
**Приклади:**
- "RaceChip GTS vs XLR vs Ultimate — який обрати?"
- "DO88 vs Wagner vs Forge intercooler — thermal camera test"
- "Akrapovic Carbon vs Titanium — вага, звук, ціна"

### 3. 🔧 Install Guide / How-To (1x/місяць)
**Тривалість:** 5-8 хвилин
**Що:** Покрокова установка
**SEO value:** Високий (люди шукають "how to install RaceChip", "DO88 intercooler installation")

### 4. ⚡ YouTube Shorts (4-8/місяць)
**Тривалість:** 15-60 секунд
**Що:** 
- Sound clips (Akrapovic)
- Before/after power (RaceChip)
- Satisfying installs (DO88, Eventuri)
- Quick tips
- Trending reformat з TikTok

## 🔍 SEO для YouTube

### Target Keywords (UA + EN)
```
"racechip відгуки" / "racechip review"
"do88 intercooler install"
"akrapovic exhaust sound"  
"brabus g wagon"
"тюнінг bmw" / "bmw tuning"
"чіп тюнінг що це" / "chip tuning explained"
"інтеркулер що це"
"eventuri intake review"
"urban automotive defender"
```

### Thumbnail Rules
- **Людське обличчя** або **крутий кадр авто** — CTR вище
- **Великий текст** — 3-5 слів максимум
- **Контрастні кольори** — жовтий/білий текст на темному
- **Before/After** split — працює завжди
- **Числа** — "+80 HP", "$5000 mod", "300°F cooler"

### Description Template
```
🏎 [НАЗВА відео]

⚡ Товари з цього відео:
🔗 [Product 1]: onecompany.global/shop/...
🔗 [Product 2]: onecompany.global/shop/...

📋 Таймкоди:
0:00 — Intro
2:00 — Що в комплекті
...

🏷 One Company — ваш гід по преміум тюнінгу
🌐 onecompany.global
📸 Instagram: @onecompany.global

#RaceChip #BMW #Tuning #OneCompany
```

## 🎨 Production Guidelines

### Камера
- 4K мінімум
- Cinematic 24fps для builds, 60fps для action
- Стабілізація обов'язкова

### Звук
- External mic для voice
- Окремий запис exhaust sound (high quality)
- Music: royalty-free cinematic (Epidemic Sound / Artlist)

### Монтаж
- Intro ≤ 30 сек (hook спочатку!)
- Motion graphics: subtle, branded
- Color grade: dark & moody, matching brand
- End screen: subscribe + next video
